Glossary

Nested elements
Nested elements must be entirely inside the parent element.
Empty elements
Empty elements have only an opening tag. No closing tag, no content.
Ordered Lists
Indicates a numbered list.
Unordered Lists
Indicates a bulleted list.
Hyperlinks
A reference to data that the user can follow by clicking or tapping.
